# LadderDraw

### 一、**使用说明：**

一个化工小工具，用于绘制化工原理中常见的精馏塔塔板阶梯图。

在打开的界面输入相平衡线、精馏线和q线相关信息，点击`开始`按钮即可生成精馏塔塔板阶梯图。

### 三、图像知识:

1. **分辨率:**

单位长度内**像素点**的数量。单位为**像素每英寸(ppi)**，如300ppi表示每英寸汉300个像素点。

2. **（总）像素：**

由一个数字序列表示的图像中的一个最小单位。

3. **（物理）尺寸：**

单位有英寸，厘米等，如10cm×10cm

1in= 2.54cm
